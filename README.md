Task 4: Multi-Page Website

Project Description

  This task involves building a simple multi-page website using HTML and CSS. The objective is to understand page navigation, relative paths, and maintaining consistency across multiple web pages.

Work Completed

  Created multiple HTML files: home.html, about.html, and contact.html.
  Maintained a consistent structure, navigation menu, and footer across all pages.
  Used relative paths to link pages and shared resources correctly.
  Reused a single external CSS file to ensure uniform styling across pages.
  Highlighted the active navigation link using CSS for better user experience.
  Organized files using a clean folder structure to support scalability.
  Tested navigation between pages to ensure there are no broken links.
